# GAS Deployment Instructions

1. Go to https://script.google.com/
2. Create a new project.
3. Replace the default code with the content of `google-apps-script.gs`.
4. Save and name your project.
5. Click "Deploy" > "Manage Deployments" > "New Deployment".
6. Select "Web App":
   - Execute as: Me
   - Who has access: Anyone
7. Copy the Web App URL and paste it in `submit.php` (replace `YOUR_GAS_WEB_APP_URL`).
